About Crop AI Platform

An Artificial Intelligence powered web application for crop classification using Google Earth Engine, Sentinel-2 satellite imagery and Machine Learning algorithms.

🌾 Project Overview

Crop AI Platform is designed to classify agricultural crops using satellite imagery and Machine Learning. Users can classify crops by selecting a province and crop type or uploading their own CSV dataset for analysis.

Random Forest

Ensemble Machine Learning model for accurate crop classification.

Support Vector Machine

Powerful classifier for separating crop classes using spectral features.

Google Earth Engine

Cloud-based satellite processing using Sentinel-2 imagery.

🚀 Technologies Used

🐍 Python
🌍 Google Earth Engine
🛰 Sentinel-2
🤖 Machine Learning
🌐 Flask
🗺 Folium Maps
📊 Pandas
💻 Bootstrap 5

📈 Classification Workflow

Input

Select province, crop or upload CSV dataset.

Satellite Data

Sentinel-2 imagery from Google Earth Engine.

AI Models

Random Forest & Support Vector Machine.

Output

Interactive map with model accuracy.

🎓 Final Year Project

This project demonstrates the integration of Remote Sensing, Artificial Intelligence and Cloud Computing for precision agriculture. It enables intelligent crop monitoring through Machine Learning models trained on Sentinel-2 satellite imagery processed within Google Earth Engine.
